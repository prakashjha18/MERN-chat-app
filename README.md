## Usage

### Install dependencies

```
cd server
npm install
cd client
npm install
```

### Run React dev server (http://localhost:3000)

```
npm start
```

### Run the back-end server (http://localhost:5001)

```
nodemon index
```

### To build for production

```
npm run build
```
